#  Central de Processos Internos - Power Clean

### 📝 Descrição do Projeto
Este projeto nasceu de uma necessidade real identificada no dia a dia da **Power Clean**. A empresa utiliza diversos formulários do Google Forms para gestão de processos, porém, o acesso a esses links era descentralizado (espalhados em grupos de WhatsApp e e-mails), gerando perda de tempo e confusão operacional.

**A Solução:** Desenvolvi uma "Landing Page" centralizadora, com interface intuitiva e responsiva, que serve como um hub para todos os formulários da empresa. Agora, o colaborador precisa memorizar apenas um único link para acessarualquer p qrocesso.

---

## 🛠️ Tecnologias Utilizadas
* **HTML5**: Estruturação semântica da página.
* **CSS3**: Estilização moderna utilizando **Flexbox** para garantir que a página seja responsiva (funcione em PCs e Celulares).
* **Git & GitHub**: Controle de versão e hospedagem via GitHub Pages.

---

## 💡 Funcionalidades
- [x] **Hub Centralizado**: 10 botões de acesso rápido aos principais processos.
- [x] **Design Responsivo**: O layout se adapta automaticamente para dispositivos móveis.
- [x] **Interface Intuitiva**: Uso de cards arredondados e efeitos de *hover* para melhor experiência do usuário (UX).
- [x] **Segurança**: Links configurados para abrir em nova aba (`target="_blank"`), mantendo a central sempre aberta.
